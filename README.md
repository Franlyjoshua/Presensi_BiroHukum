# Meeting Attendance System

A web-based meeting attendance management system developed using Laravel to streamline attendance tracking, digital signature collection, and meeting record management.

## Features

* Multi-step attendance form
* Digital signature integration
* Attendance tracking and management
* Responsive mobile-friendly interface
* Meeting participation reporting

## Tech Stack

* Laravel
* PHP
* MySQL
* Bootstrap
* JavaScript
